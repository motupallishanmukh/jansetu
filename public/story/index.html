    About Mythwoven

    Mythwoven is a love letter to the choose-your-own-adventure books of old — rebuilt as a single-page web experience using only HTML, CSS, and JavaScript. No frameworks. No backend. Just the browser and your imagination.

    • 🪄 Vanilla JS state machine for branching scenes
    • 🗺️ Path history with undo & replay
    • 🧩 Visual builder that writes its own story graph
    • 🔐 LocalStorage-based accounts (multi-user on one device)
    • 🎨 CSS Grid, custom properties, glassmorphism, parallax
    • 🔊 Web Audio API generated ambient tones
    • 💾 Auto-save, import/export, achievement system

    Built as a self-contained demo. Open index.html anywhere — it just works.
